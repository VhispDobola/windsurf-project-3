# LAN Quick Start

1. Download or clone the repository on every Windows PC.
2. Run `RUN_GAME.bat` on each machine.
3. Let the launcher install Python and requirements if needed.
4. On one PC, press `H` to host a lobby.
5. Share the host IP and port `50000`.
6. On the other PCs, press `J`, choose a slot or use `Auto`, then enter the host IP.

Notes:
- Allow Python through Windows Firewall on first launch, at least for Private networks.
- Host and clients must be on the same LAN, or on the same VPN if playing across the internet.
- For internet play, Tailscale, ZeroTier, or Radmin VPN is the easiest setup.
