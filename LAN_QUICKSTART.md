# LAN Quick Start (4 PCs)

1. Extract the project on all 4 PCs.
2. On each PC, run `RUN_LAN_MULTIPLAYER.bat`.
3. On one PC choose `Host`.
4. On the other 3 PCs choose `Client Player 2`, `Client Player 3`, and `Client Player 4`.
5. On clients, enter the host LAN IP and same port (default `50000`).

Notes:
- Allow Python through Windows Firewall on first launch.
- Host and clients must be on the same LAN/subnet.
- Use wired LAN for best latency/stability.
